Arquivos de Detalhes de Evidências
==============================

Coloque arquivos de texto nestas pastas para preencher as descrições de detalhes das evidências.
Cada arquivo deve ser nomeado com o ID de detalhe (por exemplo, 9.txt para o ID de detalhe 9).

Formato do arquivo:
- Conteúdo em texto simples para cada página
- Separe múltiplas páginas com === em uma linha própria

Exemplo (salvar como 9.txt):
---
Resumo do Caso:
28/12/2001
Elevador, Tribunal Distrital.
O ar no elevador estava com oxigênio reduzido no momento do incidente.
===
Dados da Vítima:
Gregory Edgeworth (Idade 35)
Advogado de defesa.
===
Dados do Suspeito:
Alan Guido (Idade 37)
Oficial de justiça do tribunal.
---

Pressione F5 no jogo para recarregar após fazer alterações.
